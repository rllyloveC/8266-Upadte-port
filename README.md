# 8266-Upadte-port
หัวข้อโปรเจค : ทำ AI Offline บน laptop ที่ต้องอ่านภาพได้ คำนวณได้ และถามตอบได้ดี ให้ทำงานคล้ายกับ Chat GPT
คุณสมบัติของ laptop : cpu intel 12500H, intel iris Xe, 24GB ddr4 memory
model ที่ใช้ : Qwen2-VL 7B Q4 สำหรับอ่านภาพเอกสารจากกล้อง, Qwen2.5 7B Q4 สำหรับถามตอบทั่วไป, deepseek-r1-distill-qwen-7b-q4_k_m สำหรับถามวิเคราะห์และคำนวณ
ลักษณะการทำงานของ model แต่ละตัว : Qwen2-VL จะอ่านภาพเอกสารจากกล้อง webcam ภายนอกแล้วอธิบายเป็นตัวอักษร จากนั้นจะส่งให้ Deepseek R1 คิดต่อไป, Deepseek R1 ใช้สำหรับตอบถำถามที่ต้องมีการคำนวณหรือวิเคราะห์เชิงลึก, Qwen2.5 ใช้สำหรับตอบคำถามที่ไม่มีการคำนวณหรือวิเคราห์
แหล่งข้อมูลสำหรับ model : ต้องการให้ model ค้นหาข้อมูลเองจากอินเตอร์เน็ตและบันทึกไว้เป็นไฟล์สำหรับใช้อ้างอิง โดยต้องการให้ฝึกเรื่อยๆแบบเปิดเป็นโหมดฝึกโดยเฉพาะ ไม่จำกัดเวลาการฝึก แต่ต้องการให้ได้ข้อมูลที่ถูกต้อง
